¿Qué es HTML?

HTML (Lenguaje de Marcado de Hipertexto) es el estándar para crear páginas web. Permite estructurar el contenido en la web.

Etiquetas Comunes

  • Encabezados: Usados para títulos y subtítulos.
  • Párrafos: Para el texto general.
  • Enlaces: Para crear vínculos a otras páginas.
  • Imágenes: Para insertar gráficos.
  • Listas: Para enumerar elementos.
